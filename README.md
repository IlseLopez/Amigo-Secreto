Amigo secreto 

Este proyecto es un desafío, donde se aplican conocimientos de lógica de programación; consiste en funciones donde en una lista se van almacenando nombres.

Después de tener los nombres almacenados, empieza el sorteo de cada nombre y se visualiza el amigo secreto.

Se encuentran funciones como agregarAmigo, limpiarCaja, mostrarLista, sortearAmigo.

Instalación 

Se puede clonar el repositorio con la URL y de ahí poderlo visualizar ejecutando el index.html.
